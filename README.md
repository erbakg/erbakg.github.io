# Self-Card

Personal landing site for Erbol Mederbekov — Senior Frontend / React Native engineer.

## Stack

Next.js 16 (App Router) · React 19 · TypeScript · Tailwind v4 · next-intl (EN / RU) · framer-motion · Vitest · Bun.

## Development

```bash
bun install
bun dev          # http://localhost:3000
bun typecheck
bun lint
bun run test
bun format
bun build
```

## Structure

`app → widgets → shared`. See `docs/superpowers/specs/2026-05-21-self-card-design.md` for full design.

## Deploy

Hosted on Vercel. `main` → production. No env vars required.
